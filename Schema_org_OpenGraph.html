Генератор метатегов Schema.org и Open Graph

Сформированные метатеги:


📋 Инструкция по использованию:

1. Заполните форму

Введите все необходимые данные о вашей странице. Все поля обязательны для заполнения.

2. Сгенерируйте метатеги

Нажмите кнопку "Сгенерировать метатеги" для создания кода.

3. Скопируйте код

Используйте кнопку "Скопировать метатеги" или выделите и скопируйте код вручную.

4. Вставьте в HTML

Вставьте скопированный код в секцию <head> вашего HTML-файла.

📝 Пример использования:

<!DOCTYPE html>
<html lang="ru">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Ваш заголовок</title>
    
    <!-- Open Graph метатеги -->
    <meta property="og:title" content="Ваш заголовок">
    <meta property="og:description" content="Ваше описание">
    <meta property="og:url" content="https://example.com">
    <meta property="og:image" content="https://example.com/image.jpg">
    <meta property="og:type" content="website">
    <meta property="og:site_name" content="Ваш заголовок">
    
    <!-- Schema.org JSON-LD -->
    <script type="application/ld+json">
    {
        "@context": "https://schema.org",
        "@type": "WebPage",
        "name": "Ваш заголовок",
        "description": "Ваше описание",
        "url": "https://example.com",
        "image": "https://example.com/image.jpg"
    }
    </script>
</head>
<body>
    <!-- Ваше содержимое -->
</body>
</html>

🔍 Что такое метатеги?

Open Graph

Метатеги Open Graph используются социальными сетями (Facebook, Twitter, LinkedIn) для создания красивых превью при публикации ссылок.

Schema.org

Schema.org - это структурированные данные, которые помогают поисковым системам лучше понимать содержимое вашей страницы.

Преимущества

  • Улучшенное отображение в социальных сетях
  • Лучшее SEO для поисковых систем
  • Повышение кликабельности ссылок
  • Профессиональный вид при шаринге
